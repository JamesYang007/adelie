Installation#

Users must have OpenMP installed in their system as a prerequisite. For gcc users, there is nothing to do since OpenMP is shipped with the compiler. For MacOS users using clang (default), we recommend installing through brew:

brew install libomp

Install the stable version of adelie using pip:

pip install adelie

Now, we load adelie to check that the installation was successful.

[1]:

import adelie as ad
