adelie.diagnostic.residuals#

adelie.diagnostic.residuals(*, X: MatrixNaiveBase32 | MatrixNaiveBase64, y: ndarray, weights: ndarray, offsets: ndarray, betas: ndarray | csr_matrix, intercepts: ndarray, glm: GlmBase32 | GlmBase64 | None = None)#

Computes the residuals.

The residual is given by

\[\begin{align*} \hat{r} = y - \hat{y} \end{align*}\]
Parameters:
X(n, p) Union[MatrixNaiveBase32, MatrixNaiveBase64]

Feature matrix.

y(n,) np.ndarray

Response vector.

weights(n,) np.ndarray

Observation weights.

Warning

Currently, the return value is only well-defined if weights are positive!

offsets(n,) np.ndarray

Observation offsets.

betas(l, p) Union[np.ndarray, scipy.sparse.csr_matrix]

Matrix with each row being a coefficient vector.

intercepts(l,) np.ndarray

Intercept corresponding to betas.

glmUnion[adelie.glm.GlmBase32, adelie.glm.GlmBase64], optional

GLM object. Default is None.

Returns:
resids(l, n) np.ndarray

Residuals.
