X#

Feature matrix.

X_group_norms#

Group Frobenius norm of X. X_group_norms[i] is \(\|X_{c, g}\|_F\) where \(g\) corresponds to the group index i.

X_means#

Column means of X.

abs_grad#

The \(\ell_2\) norms of grad across each group. abs_grad[i] is given by np.linalg.norm(grad[g:g+gs]) where g = groups[i] and gs = group_sizes[i].

active_sizes#

Active set size for every saved solution.

alpha#

Elastic net parameter. It must be in the range \([0,1]\).

benchmark_fit#

Fit time for a given BASIL iteration.

benchmark_invariance#

Invariance time for a given BASIL iteration.

benchmark_kkt#

KKT time for a given BASIL iteration.

benchmark_screen#

Screen time for a given BASIL iteration.

betas#

betas[i] is the (untransformed) solution corresponding to lmdas[i].

delta_lmda_path_size#

Number of regularizations to batch per BASIL iteration.

delta_strong_size#

Number of strong groups to include per BASIL iteration if strong rule does not include new groups but optimality is not reached.

early_exit#

True if the function should early exit based on training \(R^2\).

edpp_resid_0#

The residual \(y_c - X_c\beta\) where \(\beta\) is the solution at \(\lambda_\max\).

edpp_safe_hashset#

Hashset containing all the safe groups. If EDPP is not used, it contains all the variables.

edpp_safe_set#

A list of EDPP safe groups.

edpp_safe_sizes#

EDPP safe set size for every saved solution.

edpp_v1_0#

The \(v_1\) vector in EDPP rule at \(\lambda_\max\).

grad#

The true full gradient \(X_c^\top (y_c - X_c\beta)\) where \(\beta\) is given by strong_beta.

group_sizes#

List of group sizes corresponding to each element in groups. group_sizes[i] is the group size of the i th group.

groups#

List of starting indices to each group where G is the number of groups. groups[i] is the starting index of the i th group.

intercept#

True if the function should fit with intercept.

intercepts#

intercepts[i] is the intercept solution corresponding to lmdas[i].

lmda#

The regularization parameter at which the true solution is given by strong_beta.

lmda_max#

The smallest \(\lambda\) such that the true solution is zero for all coefficients that have a non-vanishing group lasso penalty (\(\ell_2\)-norm).

lmda_path#

The regularization path to solve for. The full path is not considered if early_exit is True. It is recommended that the path is sorted in decreasing order.

lmda_path_size#

Number of regularizations in the path if it is to be generated.

lmdas#

lmdas[i] is the i th \(\lambda\) regularization in the solution set.

max_iters#

Maximum number of coordinate descents.

max_strong_size#

Maximum number of strong groups allowed. The function will return a valid state and guaranteed to have strong set size less than or equal to max_strong_size.

min_ratio#

The ratio between the largest and smallest \(\lambda\) in the regularization sequence if it is to be generated.

n_threads#

Number of threads.

newton_max_iters#

Maximum number of iterations for the BCD update.

newton_tol#

Convergence tolerance for the BCD update.

penalty#

Penalty factor for each group in the same order as groups. It must be a non-negative vector.

resid#

Residual \(y_c - X \beta\) where \(\beta\) is given by strong_beta.

resid_sum#

Sum of resid.

rsq#

The true unnormalized \(R^2\) given by \(\|y_c\|_2^2 - \|y_c-X_c\beta\|_2^2\) where \(\beta\) is given by strong_beta.

rsq_curv_tol#

Early stopping rule check on curvature of \(R^2\).

rsq_slope_tol#

Early stopping rule check on slope of \(R^2\).

rsq_tol#

Early stopping rule check on \(R^2\).

rsqs#

rsqs[i] is the (normalized) \(R^2\) at betas[i].

setup_edpp#

True if EDPP setup is required, in which case, the solver will always solve at \(\lambda_\max\). See edpp_v1_0 and edpp_resid_0.

setup_lmda_max#

True if the function should setup \(\lambda_\max\).

setup_lmda_path#

True if the function should setup the regularization path.

strong_X_means#

Column means of X on the strong set.

strong_begins#

List of indices that index a corresponding list of values for each strong group. strong_begins[i] is the starting index corresponding to the i th strong group. From this index, reading group_sizes[strong_set[i]] number of elements will grab values corresponding to the full i th strong group block.

strong_beta#

Coefficient vector on the strong set. strong_beta[b:b+p] is the coefficient for the i th strong group where k = strong_set[i], b = strong_begins[i], and p = group_sizes[k]. This must contain the true solution values for the strong groups.

strong_g1#

List of indices into strong_set that correspond to groups of size 1. strong_set[strong_g1[i]] is the i th strong group of size 1 such that group_sizes[strong_set[strong_g1[i]]] is 1.

strong_g2#

List of indices into strong_set that correspond to groups more than size 1. strong_set[strong_g2[i]] is the i th strong group of size more than 1 such that group_sizes[strong_set[strong_g2[i]]] is more than 1.

strong_hashset#

Hashmap containing the same values as strong_set. It is used to check if a given group is strong or not.

strong_is_active#

Boolean vector that indicates whether each strong group in groups is active or not. strong_is_active[i] is True if and only if strong_set[i] is active.

strong_order#

Ordering such that groups is sorted in ascending order for the strong groups. groups[strong_set[i]] is the i th strong group in ascending order.

strong_rule#

True if strong rule should be used (only a heuristic!).

strong_set#

List of indices into groups that correspond to the strong groups. strong_set[i] is i th strong group. strong_set must contain at least the true (optimal) active groups when the regularization is given by lmda.

strong_sizes#

Strong set size for every saved solution.

strong_transforms#

The \(V\) from the SVD of \(X_{c,k}\) where \(X_c\) is the possibly centered feature matrix and \(k\) is an index to strong_set.

strong_vars#

The \(D^2\) from the SVD of \(X_{c,k}\) where \(X_c\) is the possibly centered feature matrix and \(k\) is an index to strong_set.

tol#

Convergence tolerance.

use_edpp#

True if EDPP should be used.

y_mean#

The mean of the response vector \(y\).

y_var#

The variance of the response vector \(y\), i.e. \(\|y_c\|_2^2\).
