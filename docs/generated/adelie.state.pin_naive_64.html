Default initialization method.

X#

Feature matrix.

active_begins#

List of indices that index a corresponding list of values for each active group. active_begins[i] is the starting index corresponding to the i th active group.

active_g1#

Subset of active_set that correspond to groups of size 1.

active_g2#

Subset of active_set that correspond to groups of size more than 1.

active_order#

Ordering such that groups is sorted in ascending order for the active groups. groups[strong_set[active_order[i]]] is the i th active group in ascending order.

active_set#

List of indices into strong_set that correspond to active groups. strong_set[active_set[i]] is the i th active group. An active group is one with non-zero coefficient block, that is, for every i th active group, strong_beta[b:b+p] == 0 where j = active_set[i], k = strong_set[j], b = strong_begins[j], and p = group_sizes[k].

alpha#

Elastic net parameter. It must be in the range \([0,1]\).

betas#

betas[i] corresponds to the solution corresponding to lmdas[i].

group_sizes#

List of group sizes corresponding to each element in groups. group_sizes[i] is the group size of the i th group.

groups#

List of starting indices to each group where G is the number of groups. groups[i] is the starting index of the i th group.

intercept#

True to fit with intercept.

intercepts#

intercepts[i] is the intercept at lmdas[i].

iters#

Number of coordinate descents taken.

lmda_path#

Regularization sequence to fit on.

lmdas#

lmdas[i] corresponds to the regularization \(\lambda\) used for the i th outputted solution.

max_iters#

Maximum number of coordinate descents.

n_threads#

Number of threads.

newton_max_iters#

Maximum number of iterations for the BCD update.

newton_tol#

Convergence tolerance for the BCD update.

penalty#

Penalty factor for each group in the same order as groups. It must be a non-negative vector.

resid#

Residual \(y_c-X\beta\) at strong_beta (note that it always uses uncentered \(X\)!).

resid_sum#

Sum of resid.

resid_sums#

resid_sums[i] is the sum of resids[i].

resids#

resids[i] is the residual at betas[i].

rsq#

Unnormalized \(R^2\) value at strong_beta. The unnormalized \(R^2\) is given by \(\|y_c\|_2^2 - \|y_c-X_c\beta\|_2^2\).

rsq_curv_tol#

Early stopping rule check on curvature of \(R^2\).

rsq_slope_tol#

Early stopping rule check on slope of \(R^2\).

rsq_tol#

Early stopping rule check on \(R^2\).

rsqs#

rsqs[i] corresponds to the unnormalized \(R^2\) at betas[i].

strong_X_means#

Column means of \(X\) for strong groups.

strong_begins#

List of indices that index a corresponding list of values for each strong group. strong_begins[i] is the starting index corresponding to the i th strong group. From this index, reading group_sizes[strong_set[i]] number of elements will grab values corresponding to the full i th strong group block.

strong_beta#

Coefficient vector on the strong set. strong_beta[b:b+p] is the coefficient for the i th strong group where k = strong_set[i], b = strong_begins[i], and p = group_sizes[k].

strong_betas#

strong_betas[i] is the state of strong_beta when the i th solution is computed.

strong_g1#

List of indices into strong_set that correspond to groups of size 1. strong_set[strong_g1[i]] is the i th strong group of size 1 such that group_sizes[strong_set[strong_g1[i]]] is 1.

strong_g2#

List of indices into strong_set that correspond to groups more than size 1. strong_set[strong_g2[i]] is the i th strong group of size more than 1 such that group_sizes[strong_set[strong_g2[i]]] is more than 1.

strong_is_active#

Boolean vector that indicates whether each strong group in groups is active or not. strong_is_active[i] is True if and only if strong_set[i] is active.

strong_is_actives#

strong_is_actives[i] is the state of strong_is_active when the i th solution is computed.

strong_set#

List of indices into groups that correspond to the strong groups. strong_set[i] is i th strong group.

strong_transforms#

List of \(V_k\) where \(V_k\) is from the SVD of \(X_{c,k}\) along the strong groups \(k\) and for possibly column-centered \(X_k\). It only needs to be properly initialized for groups with size > 1. strong_transforms[i] is \(V_k\) for the i th strong group where k = strong_set[i].

strong_vars#

List of \(D_k^2\) where \(D_k\) is from the SVD of \(X_{c,k}\) along the strong groups \(k\) and for possibly column-centered \(X_k\). strong_vars[b:b+p] is \(D_k\) for the i th strong group where k = strong_set[i], b = strong_begins[i], and p = group_sizes[k].

time_active_cd#

Benchmark time for performing coordinate-descent on the active set at every iteration.

time_strong_cd#

Benchmark time for performing coordinate-descent on the strong set at every iteration.

tol#

Convergence tolerance.

y_mean#

Mean of \(y\).

y_var#

\(\ell_2\) norm squared of \(y_c\).
